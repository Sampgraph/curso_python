# Operadores matemáticos

| Operador | Operação | Exemplo |
| -------- | -------- | ------- |
| + | Adição | 2 + 3 |
| - | Subtração | 5 - 2 |
| * | Multiplicação | 10 * 4 |
| / | Divisão | 9 / 3 |

# Estrutura de decisão: `if`
## if condition
```
if condition:
    expression1
```

## if-else condition
```
if condition:
    expression1
else:
    expression2
```

## if-elif-else condition
```
if condition:
    expression1
elif:
    expression2
elif:
    expression3
else:
    expression4
```
